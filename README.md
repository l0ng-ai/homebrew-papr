# homebrew-papr

Homebrew tap for [Papr](https://github.com/l0ng-ai/papr) — a desktop RSS reader

## Install

```bash
brew tap l0ng-ai/papr
brew install --cask papr
```

## Upgrade

```bash
brew update
brew upgrade --cask papr
```

## Uninstall

```bash
brew uninstall --cask papr
brew untap l0ng-ai/papr
```

## Supported platforms

This tap uses the official macOS release assets from Papr:

- Apple Silicon: `Papr_0.4.5_aarch64.dmg`
- Intel Mac: `Papr_0.4.5_x64.dmg`
